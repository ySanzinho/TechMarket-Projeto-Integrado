# TechMarket - Projeto Integrado

Projeto desenvolvido para a disciplina de Projeto Integrado do curso de Análise e Desenvolvimento de Sistemas.

## Tecnologias Utilizadas
- Node.js
- Express
- HTML5
- CSS3
- JavaScript
- MySQL
- Git e GitHub

## Estrutura do Projeto

- /api -> API REST de transferências financeiras
- /frontend -> Extrato bancário responsivo
- /formulario -> Validação de formulário
- /banco -> Scripts SQL e procedure

## Funcionalidades

- API de transferência com validação de saldo e UUID
- Extrato bancário responsivo com destaque de valores acima de R$ 5.000
- Procedure SQL para cálculo de saldo e listagem de transações
- Validação de CPF, data de nascimento e telefone em JavaScript

## Como Executar

### API
```bash
cd api
npm install
npm start
```

### Frontend
Abrir o arquivo `frontend/extrato.html` no navegador.

### Formulário
Abrir o arquivo `formulario/cadastro.html` no navegador.
